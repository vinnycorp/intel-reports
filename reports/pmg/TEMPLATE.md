# /pmg/ Report Telegram Template
# The cron job MUST follow this format exactly. No deviations.
# Use markdown formatting (bold with **, hyperlinks with [text](url))
# Do NOT use HTML tags like <b> — Telegram uses markdown here.

## FORMAT:

**/pmg/ Daily Report: {Month} {Day}, {Year} 🪙**
(e.g. "Feb 23, 2026" — Month is the abbreviated month name, NOT the day of week)
{N} threads analyzed · {POSTS} posts · [1]({THREAD_URL_1}) · [2]({THREAD_URL_2})

━━━━━━━━━━━━━━━━━━━━

**TOP THEMES**

1\. **{Theme Name}** ({N} mentions) | [1]({POST_URL}) · [2]({POST_URL})
{2-3 sentence summary of the theme. No emojis in theme names.}

2\. **{Theme Name}** ({N} mentions) | [1]({POST_URL}) · [2]({POST_URL})
{2-3 sentence summary.}

{...up to 5 themes}

━━━━━━━━━━━━━━━━━━━━

**ACTIONABLE SIGNALS**

⚠️ {signal} - {action}
🇨🇳 {signal} - {action}
🚨 {signal} - {action}
🌍 {signal} - {action}
📈 {signal} - {action}
⛏️ Mining mentions: {TICKER} ({N}), {TICKER} ({N})

━━━━━━━━━━━━━━━━━━━━

**CONTENT IDEAS**

- {idea}

- {idea}

- {idea}

## RULES:
- Source links use [1], [2] etc (not "source 1")
- Source links go INLINE with mentions count, separated by |
- No emojis in TOP THEMES item names
- Emojis YES in ACTIONABLE SIGNALS
- No line breaks between ACTIONABLE SIGNALS items
- Line breaks between CONTENT IDEAS items
- Use - (dash) not — (em dash) in body text
- Bold section headings: **TOP THEMES**, **ACTIONABLE SIGNALS**, **CONTENT IDEAS**
- Bold theme names within items
- Content ideas heading is just **CONTENT IDEAS** (no client name)
- Use abbreviated month names: Jan, Feb, Mar, Apr, May, Jun, Jul, Aug, Sep, Oct, Nov, Dec
